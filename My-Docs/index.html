Local / VPS Deploy

Requirements

  • Debian Based VPS (Ubuntu etc.)
  • Python 3.10+

Upgrade and Update:

sudo apt-get update && sudo apt-get upgrade -y

Installing Required Packages:

sudo apt-get install python3-pip ffmpeg -y

Setting up PIP

sudo pip3 install -U pip

Installing Node

curl -fssL https://deb.nodesource.com/setup_20.x | sudo -E bash - \ && sudo apt-get install nodejs -y \ && sudo npm i -g npm

Clone the Repository

git clone https://github.com/TEAMRIYA/RiyaMusic && cd RiyaMusic

Install tmux

sudo apt-get install tmux

Create a tmux session

Creating a tmux session will make the bot run in background, even if you close the application.

tmux new -S RiyaMusic

Setup Virtual Environment

python -m venv .venv

Activate Venv and Install Requirements

source .venv/bin/activate \ && pip3 install -U -r requirements.txt

Create .env with sample.env

cp sample.env .env
vi .env

Edit .env with your vars

Edit .env with your values. You can simply copy a config from here, paste it to your notepad, edit it, and then paste it there.

• Press I button on keyboard to start editing.

• Press Ctrl + C once you are done editing vars.

• Type :wq to save .env or :qa to exit editing.

Start the bot

bash start

Finally Exit from Tmux session

Ctrl + b d
